# Support

FableCodex is community-supported.

## Where To Ask

- Use GitHub issues for reproducible bugs, documentation problems, and focused feature requests.
- Use discussions, if enabled, for broad ideas and workflow examples.
- For security concerns, follow `SECURITY.md`.

## What To Include

For bugs:

- The command or Codex prompt you used.
- Expected behavior.
- Actual behavior.
- Relevant output, redacted for secrets.
- Your operating system and Python version when scripts are involved.

For feature requests:

- The workflow you want to improve.
- Why existing guidance is insufficient.
- The smallest example that shows the need.

## What This Project Cannot Provide

- Anthropic, Fable, OpenAI, or LiteLLM account access.
- Legal, financial, medical, or security incident response advice.
- Guarantees that prompt changes reproduce another model's capabilities.
